JsonCodec
=====================

Interfaces to serialize and deserialize PHP objects to/from JSON.

Additional documentation about this library can be found on
[mediawiki.org](https://www.mediawiki.org/wiki/JsonCodec).


Usage
-----

```php
  // FILL THIS IN
```


Running tests
-------------

```
composer install
composer test
```

History
-------
This library was first introduced in MediaWiki 1.36.0 ([dbdc2a3cd33](https://gerrit.wikimedia.org/r/c/mediawiki/core/+/641575/)). It was
split out of the MediaWiki codebase and published as an independent library
during the MediaWiki 1.40 development cycle.
